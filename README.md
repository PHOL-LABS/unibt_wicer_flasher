https://sle118.github.io/squeezelite-esp32-installer/

* The UI has been simplified.

## Related Projects

### PHL UniBT C64
PHL UniBT C64 is a Bluetooth joystick adapter for Commodore 64 (C64), VIC-20, and Amiga systems. It is powered by Bluepad32 firmware and supports modern Bluetooth controllers with low latency, including a dual joystick mode. It is compatible with Unijoysticle 2 by Ricardo Quesada: https://github.com/ricardoquesada/unijoysticle2.

### Wicer
Wicer is a PHL fork of WiC64, the Commodore 64 Wi-Fi adapter ecosystem created by the WiC64 team. The Wicer fork adds embedded SD2IEC support and includes an optimized PCB layout and design.
